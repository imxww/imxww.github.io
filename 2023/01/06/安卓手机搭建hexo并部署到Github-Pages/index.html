安卓手机搭建hexo并部署到Github-Pages

发表于2023-01-06|更新于2023-01-06
|阅读量:

前言

如你所见这是一个用手机部署的hexo博客
为什用手机部署hexo并Github Pages呢？因为方便，因为可以躺在床上弄。

准备工作安卓手机下载Termux

Termux

切换镜像

安装git

pkg install git

安装nodejs

pkg install nodejs-lts

安装vim

pkg install vim用于编写文章

部署hexo

Git Hexo主题我选的是butterfly主题

在hexo根目录下或许你可以cd hexo

其他配置和主题升级请看bytterfly

配置 _config.yml

拉到最下面去找到theme: landscape

landscape改成bytterfly

输入i修改，修改完再ESC拉到最上面去你可以按照我这样修改


修改完后按ESC输入:wp保存退出

启动hexo

这时就会得到一个地址
http://localhost:4000

配置信息

name=你的github用户名
emai1=你的邮箱
全部一起复制如下

获取密钥

输入你的短密钥，也可以留空，再输一次或许留空。

复制里面的所有内容

Github Pages

创建Github仓库我就不说了！！！！

登陆自己Github的账号，进入Settings , 找到SSH keys
Title随便填，把刚刚得到的密钥粘贴在key里面保存。
点击<>Developer settings
点击 Tokens(classic)新建一个，第一个随便填，权限全✓，保存，会得到一个密钥，复制

进入hexo

翻到最下面输入i按照我这样修改

保存退出来
安装hexo-deployer-git

会得到以下:

然后浏览器输入你的GIthub仓库域名就可以尽情的享受了
备注:其实手机Root了用MT管理器改文件是最方便最快的。

结言:因为怎么域名刚好在备案我就弄了hexo，本来是玩系统的，哈哈哈哈哈
